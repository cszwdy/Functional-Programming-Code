This function is doing quite a bit: generating random input values, checking whether they satisfy the property argument, and repeatedly shrinking a counterexample, once one is found. One advantage of defining the repeated shrinking using iterateWhile, rather than a separate while loop, is that the control flow of this piece of code stays reasonably simple.
