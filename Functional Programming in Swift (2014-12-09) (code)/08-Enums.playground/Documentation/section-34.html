Given two types, T and U, the enumeration Add<T, U> consists of either a (boxed) value of type T, or a (boxed) value of type U. As its name suggests, the Add enumeration adds together the members from the types T and U: if T has three members and U has seven, Add<T, U> will have ten possible members. This observation provides some further insight into why enumerations are called sum types.
