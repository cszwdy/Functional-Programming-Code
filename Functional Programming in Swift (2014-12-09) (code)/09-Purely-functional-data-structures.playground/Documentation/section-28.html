This function is a bit tricky. It starts by checking if the current root is marked as a member of the trie or not. If it is, the trie contains the empty key; if it is not, the result variable is initialized to the empty array. Next, it traverses the dictionary, computing the elements of the subtries — this is done by the call elements(value). Finally, the 'character' associated with every subtrie is added to the front of the elements of that subtrie — this is taken care of by the map function.
