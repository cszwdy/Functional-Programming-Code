This definition states that every tree is either:

  • a Leaf without associated values, or
  • a Node with three associated values, which are the left subtree, a value stored at the node, and the right subtree. Unfortunately, limitations in the Swift compiler prevent us from storing a recursive subtree or a generic value of type T in the Node directly. One workaround is, once again, to wrap these in a Box explicitly, which is accepted by the compiler.
