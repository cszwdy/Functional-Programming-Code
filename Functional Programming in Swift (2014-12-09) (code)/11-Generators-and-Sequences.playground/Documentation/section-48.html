The removeElement function keeps track of a variable i. When asked for the next element, it checks whether or not i is less than the length of the array. If so, it computes a new array, result, and increments i. If we have reached the end of our original array, we return nil.

We can now see that this returns all possible arrays that are one element smaller:

removeElement([1, 2, 3])
