This parser doesn't consume any tokens and returns a function that takes a character and turns it into an integer. Let's use the extremely simple input stream "3" as example. Applying the integerParser to this input yields the following sequence:

[ (A -> B, "3") ]

Applying another parser to parse the symbol "3" in the remainder (which is equal to the original input since the integerParser didn't consume any tokens) yields:

[ ("3", "") ]
